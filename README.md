## Setup Instructions

1. **Clone the repository:**
   ```
   git clone <repository-url>
   cd golang-tutorial
   ```

2. **Install dependencies:**
   ```
   go mod tidy
   ```

3. **Run the application:**
   ```
   go run cmd/main.go
   ```
